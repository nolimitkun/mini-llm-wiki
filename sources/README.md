# Sources

Raw source material the wiki draws on: papers, articles, blog posts, meeting notes, clipped web pages.

Rules (from [CLAUDE.md](../CLAUDE.md)):

- **Immutable** — files here are never edited after saving; the wiki layer in `pages/` is where synthesis happens.
- Name files descriptively: `2026-05-vaswani-attention-is-all-you-need.pdf`, `2026-07-vendor-x-pricing-page.md`.
- Pages that draw on a source list it in their frontmatter `sources:` field.
- Ingesting a new source follows the **Ingest** workflow in CLAUDE.md and gets a line in `log.md`.
